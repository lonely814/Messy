## 1. 项目概述

一个 Tampermonkey 油猴脚本，把两个旧脚本合一：

1. **HuggingFace 文件页**的多镜像下载（hf-mirror + ModelScope）。
2. **ComfyUI 缺失模型面板**的镜像下载 + 打开模型目录快捷键。

按域名分流，单文件分发，无外部依赖。

- GreasyFork 英文名称：`MultiMirror Download (HuggingFace + ComfyUI)`
- 中文别名：`多镜像下载 (HuggingFace + ComfyUI)`
- License：MIT

## 3. 架构

脚本入口按 `location.hostname` 分流：

- `huggingface.co` → 模块 A（文件页逻辑）
- `127.0.0.1:8188` / `localhost:8188` → 模块 B（ComfyUI 面板逻辑）

### 3.1 共享逻辑（顶部）

| 函数 / 常量 | 作用 |
|------|------|
| `HF_MIRROR_HOST` / `MODELSCOPE_HOST` | 镜像基础地址 |
| `COMFY_MODELS_ROOT` | **需用户配置**的本机 ComfyUI `models` 绝对路径；留空则 📂 只复制相对路径 |
| `normalizeHfPath` | 归一化 HF URL 路径 |
| `toHfMirrorUrl` / `toModelScopeUrl` | 由 HF URL 生成镜像 URL |
| `createDownloadIcon(options)` | 生成按钮。参数：`className`、`title`、`color`、`textColor`、`label`、`url`、`onClick` |
| `urlExists(url)` | HEAD 探测 + 8s 超时 + `checkedModelScopeUrls` Map 缓存，决定 MS 按钮是否显示 |
| `extractDirectory(btn)` | 从按钮附近文本识别模型子目录类型 |
| `openModelFolder(directory, filename)` | 📂 打开 `file://` 目录 + 复制路径到剪贴板 |

### 3.2 模块 A — HuggingFace 文件页

- 选择器：`a[href*="/resolve/"]` 且行内含体积文本（如 `1.2 GB`）判定为下载链接。
- 官方下载图标：`h-5 w-5` 且含 `<svg>` 的 div。
- 在官方图标后注入 `HF` / `MS` 按钮；用 `dataset` 标记防重复注入。

### 3.3 模块 B — ComfyUI 缺失模型面板

- 官方下载按钮：`[data-testid="missing-model-download"]`（BUTTON，无 href）。
- **模型 URL 不在按钮子树里**，而在画布 `comfy-markdown-content` / `lg-node-widget` 节点的 hf 锚点中。
- `buildAnchorMap()`：遍历 `a[href*="huggingface.co"]`，取 pathname 末段解码作 key → href。
- `resolveUrl(btn, anchorMap, assigned)`：文件名精确一对一配对（`assigned` Set 去重）；兜底取未分配候选。
- 按钮顺序：官方 → HF(黄) → MS(紫，仅存在时) → 📂(绿，整行最右侧)。

## 4. 关键实现决策

- **不依赖 Vue**：ComfyUI 生产构建剥离了 Vue 实例（`__vueParentComponent` 不可用），全部从 DOM 读取。
- **独立子树配对**：面板按钮与画布节点是独立 DOM，不能靠爬父节点找锚点，改用文件名映射。
- **hf-mirror 防盗链绕过**：下载按钮是真实 `<a href target="_blank" rel="noreferrer noopener">`，无 Referer 时 hf-mirror 才返回附件（否则返回 HTML 导致页面跳转/弹"离开此网站"）。
- **IDM 集成限制**：IDM 不捕获 `target="_blank"` 新标签页下载，故左键走浏览器默认下载；走 IDM 请右键按钮 → "Download with IDM"。同页/iframe 方式会被 hf-mirror 防盗链拦截，不可行。
- **按钮配色（品牌主题）**：hf-mirror = 黄 `#f5b301`（深色字 `#1f2937`），ModelScope = 紫 `#7c3aed`（白字），文件夹 = 绿 `#16a34a`。文字标签 `HF` / `MS` / `📂` 区分图标，不只靠颜色。
- **下载落点**：浏览器安全限制，网页下载只能落浏览器默认目录，非 ComfyUI `models`；用户需手动移动。

## 5. 配置项

脚本顶部：

```js
const COMFY_MODELS_ROOT = '';
```

填本机 ComfyUI `models` 绝对路径（Windows 用双反斜杠或正斜杠），📂 才能直接打开本地文件夹。

## 6. 已知问题与限制

1. **`file://` 打开被拦截**：Chrome 通常禁止 http 页面打开 `file://`，📂 打开可能失败，但剪贴板已复制路径（兜底）。Firefox 一般可正常打开。
2. **端口固定 8188**：换端口需改两处 `@match`（127.0.0.1 / localhost）。
3. **子目录靠文本识别**：`DIR_TYPES` 列表匹配按钮附近文本；识别不到回退到 `models` 根目录。
4. **MS 显示依赖 HEAD 探测**：网络慢时按钮出现有延迟（8s 超时）。
