# 无限画布 · 性能与视觉增强

## 这是什么画布？

[Infinite Canvas](https://github.com/hero8152/Infinite-Canvas) 是一个开源 AI 生图工作流编辑器。界面像一块无限大的白板，拖入各种节点（图片、视频、提示词、LLM、生成器……），连线组成工作流，一键跑图出图。

节点多了、图片多了，画布会变卡。这个脚本就是来解决这个问题的。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. [点此安装脚本]()（待上传）
3. 打开你的画布页面，刷新即可

## 快捷键

| 按键 | 功能 | 说明 |
|------|------|------|
| `G` | 网格 | 隐藏/显示背景网格 |
| `T` | 性能 | 去掉阴影毛玻璃，拖动更丝滑 |
| `H` | 隐藏 | 隐藏所有图片/视频，纯浏览节点结构 |
| `L` | 连线 | 隐藏/显示连接线 |
| `O` | 列表 | 输出节点切换列表布局，文件名一目了然 |
| `?` | 帮助 | 隐藏/显示底部快捷键提示 |

油猴菜单里也有对应开关，状态自动保存，刷新不丢。

## 性能优化

**LOD 多级缩放** — 缩小画布时自动降级渲染：
- L1：图片模糊+半透明
- L2：完全隐藏图片，只留节点框
- L3：节点缩成小方块，只显示标题

**content-visibility** — 图片/视频节点按需渲染，画外的不占渲染资源。

**拖拽节流** — 拖拽画布或节点时临时隐藏所有图片，拖完恢复。

**内存清理** — 删除节点时自动释放图片内存，长时间使用不臃肿。

## 视觉增强

**选中发光** — 选中节点多层发光，浅色紫色 / 深色蓝色。

**连线圆角** — 圆头圆角连接，粗细透明度可调，激活时带辉光。

## 列表模式（输出节点）

输出节点默认是图片网格，图片一多找起来费劲。按 `O` 一键切换列表：

```
[缩略图] [文件名.......................] [×]
```

每张图片一行，文件名在中间，删除按钮在最后，不容易误触。
