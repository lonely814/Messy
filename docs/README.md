# LibTV Canvas Boost

> liblib.tv / iblib.tv 画布增强脚本 — 性能优化、视觉美化、智能链导航

## 功能一览

### ⌨️ 快捷键

| 键 | 功能 |
|----|------|
| `G` | 切换网格背景显隐 |
| `T` | 性能模式：关闭动画/模糊/阴影，降低节点不透明度 |
| `H` | 隐藏所有图片，只留边框 |
| `L` | 隐藏所有连线 |
| `C` | 自动全链路模式：选中任意节点，自动高亮上下游所有关联节点和连线，点击空白清除 |
| `F` | 节点搜索：按名称或 ID 模糊搜索，点击跳转 |
| `P` | 提示词工具：模板库 / AI 增强 / 设置 |
| `X` | 专注模式：隐藏工具栏/侧边栏，画布全屏 |
| `?` | 底部快捷键提示显隐 |
| `Esc` | 关闭搜索 / 关闭提示词工具 / 清除链高亮 / 关闭诊断面板 |

### 🎨 视觉美化

- **选中节点** — 多层发光轮廓 + 底部渐变光晕条，与画布圆角对齐
- **连线** — 默认淡出到 10% 不透明度，链高亮时彩色发光
- **FPS 面板** — 紧凑小药丸，显示帧率 / 缩放 / 节点数 / 功能状态，可拖拽
- **搜索框** — 玻璃模糊效果
- **帮助栏** — 底部滑入/滑出动画
- **全部节点** — box-shadow / opacity 平滑过渡

### 🎨 主题 (P → 🎨 主题)

- **6 套预设** — 靛蓝 / 翡翠 / 玫瑰 / 琥珀 / 天蓝 / 紫色，一键切换
- **自定义颜色** — 颜色选择器自由选色，实时预览
- 所有主题持久化到 localStorage

### 📋 提示词工具 (P)

- **模板库** — 内置5组默认提示词模板（产品摄影/电影镜头/吉卜力风/产品展示/风光延时），支持分类筛选、新增、编辑、删除
- **AI 增强** — 两种模式：🇨🇳 中文增强 / 🇺🇸 English Enhance，调用 OpenAI 兼容 API 润色扩写提示词
  - 默认对接 DeepSeek (`https://api.deepseek.com/chat/completions`, `deepseek-v4-flash`)
  - 从画布节点获取内容后，增强结果点击直接写回节点
- **设置** — 可自定义 API 地址、Key、模型
- 画布右下角有 📋 浮动按钮可拖拽，快捷键 `P` 开关

### 📦 其他

- 所有开关状态自动保存到 localStorage，刷新后保持
- Tampermonkey 菜单命令：性能模式 / 隐藏图片 / 隐藏连线 / 隐藏网格 / 诊断

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开本脚本的 [Greasy Fork 页面](替换为你的GreasyFork链接)
3. 点击「安装此脚本」
4. 打开 liblib.tv 或 iblib.tv 的画布页面，自动生效
